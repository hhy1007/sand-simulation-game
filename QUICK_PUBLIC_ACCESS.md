## 💻 **方法2: JSFiddle 在线运行**

### **步骤：**
1. **访问**: https://jsfiddle.net
2. 将代码分到四个区域：
   - HTML (包含CSS链接)
   - CSS (样式代码)
   - JavaScript (游戏逻辑)
   - External Resources (添加外部库)

## 🚀 **方法3: Glitch 实时托管**

### **步骤：**
1. **访问**: https://glitch.com
2. 点击 **New Project** → **Import from GitHub**
3. 输入GitHub仓库URL
4. 点击 **OK**

### **访问地址：**
```
https://your-project-name.glitch.me
```

## 📦 **方法4: StackBlitz Web容器**

### **步骤：**
1. **访问**: https://stackblitz.com
2. 点击 **Start a new Project** → **Static**
3. 上传项目文件
4. 自动获得预览链接

## 🔗 **方法5: 使用GitHub直接预览**

即使没有启用Pages，也可以使用GitHub的原始文件预览：

### **临时链接格式：**
```
https://raw.githack.com/YOUR-USERNAME/sand-simulation-game/main/public/index.html
```

或使用：
```
https://htmlpreview.github.io/?https://github.com/YOUR-USERNAME/sand-simulation-game/blob/main/public/index.html
```
